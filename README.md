# Alacritty.icns

一组 Alacritty 图标，包括

* 白色
* 粉色
* 绿色
* 蓝色

## 如何替换默认的应用图标

* 首先去 release 页面下载任一[图标](https://github.com/lewangdev/Alacritty.icns/releases/tag/v0.0.1)
* 在应用文件夹中找到 Alacritty.app 选中后快捷键 command + i 打开显示简介
* 然后将下载的 icns 文件拖动到左上角图标上，即可完成替换
* 最后重新应用图标就会自动更新
